# GPSLive Device Tracker Add-on

Vehicle tracking integration for Home Assistant using GPSLive API.

## About

This add-on integrates GPSLive vehicle tracking system with Home Assistant, creating device tracker entities for your vehicles.

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the "GPSLive Device Tracker" add-on
3. Configure the add-on (see Configuration section)
4. Start the add-on

## Configuration

```yaml
debug_logging: false
```

### Option: `debug_logging`

Enable debug logging for troubleshooting.
