## Configuration Options

### `debug_logging` (optional)
- **Type**: boolean
- **Default**: `false`
- **Description**: Enable verbose debug logging for troubleshooting

## Getting Started

1. Install the add-on from the add-on store
2. Configure your preferences in the Configuration tab
3. Start the add-on
4. Check the logs to ensure everything is running correctly

## Troubleshooting

Enable `debug_logging` in the configuration to get more detailed logs.
