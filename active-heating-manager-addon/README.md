# Active Heating Manager Add-on

Smart heating management for Home Assistant.

## About

This add-on provides intelligent heating management capabilities for your Home Assistant installation.

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the "Active Heating Manager" add-on
3. Configure the add-on (see Configuration section)
4. Start the add-on

## Configuration

```yaml
debug_logging: false
```

### Option: `debug_logging`

Enable debug logging for troubleshooting.
